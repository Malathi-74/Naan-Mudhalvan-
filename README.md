# Customer Support Chatbot

A simple web-based **Customer Support Chatbot** built with **Flask**.  
It provides automated responses to common customer queries using a **rule-based knowledge base**.

## Features
- Handles **greetings**, **goodbye**, and **help** messages.
- Provides information on **account**, **orders**, **products**, and **payments**.
- Built-in **fallback responses** for unknown queries.
- Responsive **frontend chat interface**.
- Easy to deploy locally using Flask.


## Technologies Used
- **Backend:** Python, Flask, Flask-CORS
- **Frontend:** HTML, CSS, JavaScript
- **Other:** Rule-based knowledge system
